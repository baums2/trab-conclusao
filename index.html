Projeto Controle Financeiro

O Projeto Cotrole Financeiro foi criado como trabalho de conclusão do curso Técnico em Informática para Internet, esse curso é encontrado Gratuitamente no Colégio Estadual protásio Alves, na construção do sistema foram utilizadas as ferramentas listadas a baixo, juntamente com a orientação dos professores do curso, mesmo que simples o sistema está bem funcional, com o decorrer dos meus estudos vou atualizando e aprimorando o mesmo.







Funcionabilidades

  • Cadastro de perfil
  • Autenicação com Sessão
  • Inclusão de recitas/Despesas
  • Exclusão e Edição das Receitas/Despesas
  • Balanço Mensal
  • Balanço Anual

Ferramentas utilizadas

  • HTML/CSS
  • PHP 5
  • Java Script
  • Mysql
  • Bootstrap interface responsiva
